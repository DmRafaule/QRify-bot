# QRify

## Description
QRify bot made for generating QR codes and reading an existing ones from photo and transforming them into readable text.
More detailed description you may find on the website of the developer [QR to text and vise versa](https://timthewebmaster.com/en/tools/)

## Basic features
* Convert a input text into QR code
* Convert images with QR code into text
* Inline mode, this means you can use this bot in any chat to be able to covert your text into QR code
